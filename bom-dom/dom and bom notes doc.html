## `setTimeout` + `setInterval`
______________________________________________
- What is a callback function?
is a function that returns a value. Invoking a function "sayHello()" instead of referencing the code example "sayHello"
______________________________________________
- What do `setTimeout` and `setInterval` allow us to do? What are some examples of this?
______________________________________________
- What is the difference between `setTimeout`, `setInterval`, and `clearInterval`? setInterval and setTimeout let us execute code after a delay, or repeatedly with a delay.
Example: setTimeout passing function that will be called for us.
setTimeout(function() {
console.log("hi")
} 2000 )
Will return "Hi" after 2000 milliseconds = 2 seconds
or
setTimeout(sayHello,4000) means, after 4 seconds call sayHello function.
Example: setInterval
setInterval(function() {
console.log("inside setInterval callback")
} 1000 )
Will return "inside setInterval callback" every 1000 milliseconds = 1 seconds
clearInterval will clear after a certain period What if I wanted to count down every 5 seconds with a 1 second delay: example:
i = 5
var intervalId = setInterval(function(){
console.log(i);
i -= 1;
if (i <= 0) {
console.log("blast off!")
clearInterval(intervalId);
}
}, 1000);
references our global i variable
